# Shifty ⏱️

מעקב אישי אחר שעות עבודה, משמרות והערכת שכר. מקומי, מהיר, ללא הרשמה — הכול נשמר במכשיר ועובד אופליין.

> חלופה נקייה, מהירה ויפה יותר ל"אחראי משמרת".

## הפעלה

**הכי פשוט:** דאבל-קליק על `start.bat` ואז פתח את `http://localhost:4173` בדפדפן.

או, לבדיקה מהירה — אפשר פשוט לפתוח את `index.html` בדפדפן (במצב הזה ה־Service Worker וההתקנה כ־PWA לא יפעלו, אבל כל שאר האפליקציה כן).

## מבנה

```
index.html                 שלד האפליקציה (RTL, עברית)
manifest.webmanifest       הגדרות PWA
sw.js                      Service Worker (אופליין)
assets/css/styles.css      מערכת עיצוב (בהיר ברירת מחדל + כהה נקי)
assets/js/format.js        עיצוב תאריך/שעה/כסף בעברית
assets/js/store.js         שכבת נתונים (localStorage) + מחזור חיי משמרת
assets/js/calc.js          מנוע חישוב שעות ושכר
assets/js/app.js           בקר ותצוגות (עכשיו / משמרות / הגדרות)
assets/icons/icon.svg      אייקון
```

## מה כבר עובד (הלולאה המרכזית)

- **עכשיו** — טבעת התקדמות חיה, טיימר זמן עבודה, התחל/סיים משמרת, התחל/סיים הפסקה, שכר מצטבר בזמן אמת.
- **משמרות** — רשימה לפי מחזור חודשי, ניווט בין תקופות, סיכום שעות/משמרות/שכר, עריכה והוספה ידנית.
- **הגדרות** — שכר לשעה, שעות נוספות (125%/150%), הפסקה אוטומטית, שבת/חג, שעות לילה, נסיעות, יום תחילת חודש, יעדים, ערכת נושא, גיבוי/שחזור.

## חישוב שכר

- שעות נטו = ברוטו פחות הפסקות (כולל הפסקה אוטומטית למשמרות ארוכות).
- שעות רגילות עד הסף היומי, אחר כך שעות נוספות מדורגות (125% לשעתיים הראשונות, 150% אחר כך).
- שבת/חג: כל המשמרת במכפיל החג.
- שעות לילה: תוספת על שעות בחלון הלילה (אופציונלי).
- נסיעות: החזר ליום עבודה, עם תקרה חודשית אופציונלית.
